Saara dLAB

dLAB, Udupi Overview dLAB is complete Medical Laboratory software that full fills all necessities of Clinical Lab Management. dLAB has special feature of Automatic Entry, Where Result of a Test automatically loaded to dLab software from HL7 compatible machines (version 3).Health Level Seven / HL7 protocol is used to transfer clinical data between software applications.

Features

Features include reception printing the test result reports, Inventory Management and many more at one place with fully expertise and error free operations

• Complete and automatic operating of reception.
•Test can be created as Single test, Tests in Group or Test Profile.
• Automatic and error free report printing of different kind of test results.
• Test results based on formula and templates.
• Maintaining the inventory section of your lab.
• Built in backup and restore facilities.
• LAN compatible.

Reports on your fingertips:

Patient/Doctor can view the reports in their mobile.

• Report as text message
• Reports in PDf.

Master:

• Test, the medical lab software includes all kinds of test that are currently used.
• Facilities to add any new kind of test.
• Test type, all kinds of sub test types are also provided with customizing facility.
• Service, you can define any new services that are provided by you and its charges.
• Products, you can also hold the details of your products if you sale them.

Reception:

A more advance reception for all things to be done automatically.

• Creating a new patient entry.
• View and modify all old patient details.
• Select any patient, calculate his bills for the different facilities provide by you like lab test charge, sonography.
• Change, ECG, Charge, X-ray charge, ambulance charge any additional service charge.

Results & Reports:

• Male & Female reference range can be set for every result.
• Formula based result calculation.
• Results can be entered in values on can be generated as templates which can be edited in Word.

Inventory Master:

Medical laboratory software also provides facility to maintain complete inventory like:

• Sale / Purchase data entry.
• Sale / Purchase Return.

Inventory Reports:

All kinds of inventory reports can be generated within seconds and printed by dLab software.

• Stock Report
• Sale / Purchase Book.
• Stock Ledger.
• Sale / Purchase Return Book.
